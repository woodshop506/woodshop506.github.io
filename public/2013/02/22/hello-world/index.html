update: Jekyll-Bootstrap was okay, but my blog is now The New Hotness

Easy peasy

  • Start new Github repo, maybe with Jekyll-Bootstrap
  • Find the Readable Theme and tweak styles/markup
  • Tell yourself that you’ll compile Bootstrap without the unnecessary modules, then motivate yourself using constructive language
  • Set A/CNAME records to serve the site from your own domain (if user.github.io isn’t to your liking)
